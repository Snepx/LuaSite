Aprender Lua - Do Zero ao Avançado

1. Introdução à Lua

Lua é uma linguagem de programação leve e rápida, amplamente utilizada em jogos, aplicações embarcadas e scripts. Sua sintaxe é simples e flexível, permitindo fácil aprendizado e implementação.

2. Variáveis e Tipos de Dados

Lua possui tipos de dados como números, strings, booleanos, tabelas, funções e threads. As variáveis são declaradas usando a palavra-chave local.


-- Declaração de variáveis
local numero = 42
local nome = "Lua"
local isActive = true

3. Controle de Fluxo

Usamos condicionais e loops para controlar o fluxo do programa. As estruturas condicionais incluem if, else e elseif.


-- Estruturas condicionais
local idade = 18
if idade >= 18 then
    print("Você é maior de idade.")
else
    print("Você é menor de idade.")
end

-- Looping
for i = 1, 5 do
    print("Contagem:", i)
end

4. Funções

Funções são blocos de código que podem ser reutilizados. Elas podem receber parâmetros e retornar valores.


-- Definindo uma função
function calcularArea(base, altura)
    return (base * altura) / 2
end

-- Chamando a função
local area = calcularArea(10, 5)
print("Área do triângulo:", area) -- Saída: 25

5. Tabelas

Tabelas são a estrutura de dados mais poderosa em Lua, permitindo armazenar dados de forma complexa. Elas podem ser usadas como arrays, dicionários ou objetos.


-- Criando uma tabela
local estudante = {
    nome = "Maria",
    idade = 20,
    notas = {8, 9, 10}
}

-- Acessando os dados da tabela
print("Nome:", estudante.nome)
print("Idade:", estudante.idade)
print("Nota 1:", estudante.notas[1]) -- Saída: 8

6. Metatables

Metatables permitem a implementação de herança e comportamentos personalizados em tabelas.


-- Usando metatables para implementar herança
local animal = {}
animal.__index = animal

function animal:andar()
    print("O animal está andando.")
end

local cachorro = setmetatable({}, animal)
cachorro:andar() -- Saída: O animal está andando.

7. Manipulação de Strings

Lua tem funções integradas para manipular strings de maneira eficaz.


local frase = "Aprendendo Lua é divertido!"
local primeiraPalavra = string.match(frase, "%S+")
print("Primeira palavra:", primeiraPalavra) -- Saída: Aprendendo

8. Módulos e Bibliotecas

Você pode dividir seu código em módulos, facilitando a reutilização e organização.


-- Criando um módulo em Lua
local meuModulo = {}

function meuModulo.soma(a, b)
    return a + b
end

return meuModulo

-- Usando o módulo
local soma = require("meuModulo")
print("Resultado da soma:", soma.soma(5, 10)) -- Saída: 15

9. Tratamento de Erros

Lua possui um sistema de tratamento de erros que permite capturar e lidar com exceções.


-- Tratamento de erros
local status, err = pcall(function()
    error("Um erro ocorreu!")
end)

if not status then
    print("Erro:", err)
end

10. Funções Úteis em Lua

Aqui estão dez funções úteis que você pode usar em Lua:

10.1. math.abs(x)

Retorna o valor absoluto de x.


print("Valor absoluto de -5:", math.abs(-5)) -- Saída: 5

10.2. math.sqrt(x)

Retorna a raiz quadrada de x.


print("Raiz quadrada de 16:", math.sqrt(16)) -- Saída: 4

10.3. table.insert(t, value)

Insere um valor em uma tabela t.


local numeros = {1, 2, 3}
table.insert(numeros, 4)
print("Tabela após inserção:", table.concat(numeros, ", ")) -- Saída: 1, 2, 3, 4

10.4. table.remove(t, pos)

Remove o elemento na posição pos da tabela t.


local frutas = {"maçã", "banana", "laranja"}
table.remove(frutas, 2)
print("Frutas após remoção:", table.concat(frutas, ", ")) -- Saída: maçã, laranja

10.5. string.len(s)

Retorna o comprimento da string s.


print("Comprimento da string 'Lua':", string.len("Lua")) -- Saída: 3

10.6. string.lower(s)

Converte a string s para minúsculas.


print("Minúsculas de 'LUA':", string.lower("LUA")) -- Saída: lua

10.7. string.upper(s)

Converte a string s para maiúsculas.


print("Maiúsculas de 'lua':", string.upper("lua")) -- Saída: LUA

10.8. math.random([m[, n]])

Retorna um número aleatório entre m e n. Se apenas n for fornecido, retorna um número aleatório entre 1 e n.


math.randomseed(os.time()) -- Inicia o gerador de números aleatórios
print("Número aleatório entre 1 e 10:", math.random(1, 10))

10.9. os.date(format)

Retorna a data e hora atual, formatada de acordo com o format especificado.


print("Data atual:", os.date("%d/%m/%Y %H:%M:%S"))

10.10. coroutine.create(f)

Cria uma nova coroutine que executa a função f.


local co = coroutine.create(function()
    for i = 1, 5 do
        print("Contagem na coroutine:", i)
        coroutine.yield() -- Pausa a execução
    end
end)

coroutine.resume(co)
coroutine.resume(co)

11. Exemplos de Uso em Jogos

Aqui estão alguns exemplos de código em Lua que são comumente usados em jogos:

11.1. Movimentação de Personagem

Um exemplo simples de movimentação de um personagem em um jogo 2D.


-- Variáveis de posição
local posX, posY = 0, 0
local velocidade = 5

-- Função para mover o personagem
function mover(direcao)
    if direcao == "cima" then
        posY = posY + velocidade
    elseif direcao == "baixo" then
        posY = posY - velocidade
    elseif direcao == "esquerda" then
        posX = posX - velocidade
    elseif direcao == "direita" then
        posX = posX + velocidade
    end
    print("Posição do personagem:", posX, posY)
end

-- Simulando movimentos
mover("cima")
mover("direita")

11.2. Colisão entre Objetos

Exemplo de detecção de colisão entre dois retângulos.


-- Função para verificar colisão
function verificarColisao(rect1, rect2)
    return not (rect1.x > rect2.x + rect2.largura or
                rect2.x > rect1.x + rect1.largura or
                rect1.y > rect2.y + rect2.altura or
                rect2.y > rect1.y + rect1.altura)
end

-- Definindo dois retângulos
local jogador = {x = 10, y = 10, largura = 50, altura = 50}
local inimigo = {x = 30, y = 30, largura = 50, altura = 50}

-- Verificando colisão
if verificarColisao(jogador, inimigo) then
    print("Colisão detectada!")
else
    print("Sem colisão.")
end

11.3. Gerenciamento de Estados do Jogo

Um exemplo simples de gerenciamento de estados, como menu, jogo e pausa.


-- Estados do jogo
local estadoDoJogo = "menu"

-- Função para mudar o estado
function mudarEstado(novoEstado)
    estadoDoJogo = novoEstado
    print("Estado do jogo:", estadoDoJogo)
end

-- Mudando estados
mudarEstado("jogo")
mudarEstado("pausa")

11.4. Sistema de Inventário

Um exemplo básico de como gerenciar um inventário em um jogo.


local inventario = {}

-- Função para adicionar item
function adicionarItem(item)
    table.insert(inventario, item)
    print(item .. " adicionado ao inventário.")
end

-- Função para mostrar inventário
function mostrarInventario()
    print("Inventário:")
    for i, item in ipairs(inventario) do
        print(i, item)
    end
end

-- Adicionando itens
adicionarItem("Espada")
adicionarItem("Poção")
mostrarInventario()

11.5. Sistema de Pontuação

Um exemplo simples de como implementar um sistema de pontuação.


local pontuacao = 0

-- Função para adicionar pontos
function adicionarPontos(pontos)
    pontuacao = pontuacao + pontos
    print("Pontuação atual:", pontuacao)
end

-- Adicionando pontos
adicionarPontos(10)
adicionarPontos(5)
